# Wishler

Этот репозиторий содержит проект ASP .NET6 Core - веб приложение, которое позволяет пользователям создавать доски, колонки и ячейки в них, которые можно перетаскивать. Также в приложении есть функционал создания групп, приглашения участников и распределения желаний внутри группы.

# Как запустить проект?

1) Скопировать репозиторий себе на ПК
2) Открыть папку с репозиторием через VisualStudio, Rider или любую другую IDE, поддерживающую язык программирования C#
3) Запустить SQL Server Managment Studio
4) Создать базу данных и скопировать её название
5) В файле ```appsetting.json``` изменить значение ```LAPTOP-UJP9QTM1``` на ранее скопированное
6) Запустить проект в вашей IDE
